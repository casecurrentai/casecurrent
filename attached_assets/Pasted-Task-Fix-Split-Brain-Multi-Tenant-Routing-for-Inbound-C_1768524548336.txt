- If found: generate TwiML <Connect><Stream> and include Parameters:

Step 2: Stream handler hardening (server/telephony/twilio/streamHandler.ts)
- On Twilio "start" event:
  - Extract customParameters.orgId and customParameters.toNumber.
  - Normalize toNumber to E.164.

  - If finalOrgId missing: log [TENANT_MISSING] and abort (do not create orphan leads)
